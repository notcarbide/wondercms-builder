# wondercms-builder

## Description

Build a zip archive for WonderCMS so it can work without any build step.

## Requirements

Needs to have `git`, `yarn`, `zip`, `grunt-cli` and `composer` in `$PATH`.
Also requires `php >= 7.1.3` with `curl`, `mbstring` and `zip` modules.

## Usage

~~~bash
./build.sh
~~~
